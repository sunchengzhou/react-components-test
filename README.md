# webpack React demo


**项目运行**
```javascript
$ git clone https://github.com/tsrot/react-zhihu.git
$ cnpm install  //安装依赖文件
$ npm run build   //文件打包
$ npm run start   //实时刷新

```
### 说明  description
> 本分支仅提供最简单的react骨架 方便测试react新特性等

> This branch only provides the simplest react skeleton for testing new react features
